# Technology Stack

## Platform
- **Operating System**: Windows (win32)
- **Shell**: PowerShell/CMD
- **IDE**: VSCode with Kiro AI integration

## Development Environment
- Kiro AI assistant for code generation and assistance
- VSCode extensions and settings for enhanced productivity

## Common Commands
Since this is a fresh project, specific build/test commands will depend on the technology stack chosen. Common patterns include:

### Node.js Projects
```bash
npm install          # Install dependencies
npm run dev          # Start development server
npm test             # Run tests
npm run build        # Build for production
```

### Python Projects
```bash
pip install -r requirements.txt    # Install dependencies
python -m pytest                   # Run tests
python main.py                     # Run application
```

### General Development
```bash
git init                           # Initialize git repository
git add .                          # Stage changes
git commit -m "message"            # Commit changes
```

## Configuration
- VSCode settings configured in `.vscode/settings.json`
- Kiro steering rules in `.kiro/steering/` directory
- MCP (Model Context Protocol) disabled by default
